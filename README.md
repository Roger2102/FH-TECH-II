PROJETO FH TECH - DESENVOLVIMENTO WEB

Realizei o projeto FH TECH utilizando o editor Visual Studio Code (VSCode) e aplicando as principais tecnologias da web: HTML, CSS e JavaScript. 

COMO FOI FEITO: 
- Estruturei todo o site em HTML, organizando seções como cabeçalho, menu responsivo, apresentação, especialidades, sobre, portfólio de trabalhos, formulário de contato e rodapé. 
- Desenvolvi o CSS para criar um design moderno e responsivo, com animações, paleta de cores personalizada e foco na experiência do usuário. 
- Implementei o JavaScript para dar interatividade ao menu mobile e tornar a navegação dinâmica e intuitiva. 

O resultado é um site institucional completo, responsivo e funcional, que demonstra minha capacidade de unir design, usabilidade e programação em um projeto real. 

Esse trabalho reforça minha experiência em desenvolvimento front-end e mostra que estou preparado para criar soluções digitais que entregam valor e profissionalismo.
